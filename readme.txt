With this plugin you can choose different theme and content for your site mobile visitors.

== Description ==

This plugin creates an environment for users through which plugin users can choose a separate theme and content for visitors who enter via mobile.
With the WP Mobili plugin, you can use the pwa features without any programming knowledge on your wordpress site.

Also, the WP Mobili plugin has various and free templates for mobile mode that can be easily installed with one click.

== Frequently Asked Questions ==

**How do I install WP Mobili?**

To install the WP Mobili, follow the steps below:
From your WordPress dashboard -> Go to Plugins -> Click on 'Add new'-> In the Search field, enter WP Mobili and choose WP Mobili.
Press install -> After installation, click Activate.

**Does WP Mobili work with all the themes?**

WP Mobili works all the themes that respect the coding standards of WordPress set by its Codex. It is recommended to use WP Mobili's [Hello Mobili](https://wp-mobili.com/theme/hello-mobili/), a lightweight, to enjoy full flexibility when using WP Mobili, and optimize your experience.

**Is WP Mobili compatible with Page builders?**

WP Mobili and Page Builders work seamlessly together. As a user, you can easily decide which editor to use at every point while editing your site.

**Does it work with other WordPress plugins? **

It works with almost all the plugins. If you experience an incompatibility issue, please report it to us and to the plugin that conflicts with WP Mobili.

**Will WP Mobili slow down my website?**

The mobile plugin is programmed as standard and the mobile mode template is loaded only when the user enters your site with the mobile.
